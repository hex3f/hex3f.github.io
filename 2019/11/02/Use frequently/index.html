这个页面将会存放一些我经常使用的工具、插件或网站。
This page will store some tools, plugins, or websites that I use often.

Website

  • Model download website
    sketchfab
    里面有很多不错的免费模型，也有很好看的付费模型可以下载。对于3D游戏或者建模开发有帮助。
  • character model Animation
    mixamo
    这个网站mixamo目前是被adobe收购的，主要用于人物的动画。里面的动作非常非常多，而且可以免费下载。
  • Art images
    Artstation
    里面有很多艺术的图，个人创造的图可以提供给美术参考
  • Js library
    Threejs
    一个庞大的JS库，里面有很多特效都很强大
  • Shader
    Shadertoy
    Glslsandbox
    Thebookofshaders
    Shader例子跟教程

Unity

  • LitJSON (json plugin)
    Click here to download -Using .NET Standard 2.0 Compiled on 2019-11-2
    Related Links Official_API Github
    这是一个用于解析json的插件，虽然在unity分类里面，但是只要是C#就可以使用。我使用的环境是unity2018，版本过低会导致使用不了。
  • Lipsync Pro
    ASSETSTORE
    这是unity一个做唇形同步的插件。

Model

  • blender 模型制作-免费开源的模型制作软件，非常强大。

Character

  • iclone 动画制作-软件收费，导出也要收费。
  • character creator 3D人物制作软件-较为方便，缺点是要收费。
  • daz 3D 3D人物制作-较为复杂，但是功能很多，可扩展性也广，教程也多，而且还免费。

Photo

  • aseprite 像素图片制作-界面非常简单，而且不需要花费很多时间学习。
  • materialize 游戏贴图制作

Work

  • Teambition 这个是用于团队协作的，有wiki、协作文档等，一起开发的时候十分有用，阿里巴巴也是使用这个
  • Git 可以控制开发的版本，资源上传给远处的伙伴，合作开发很有用！
