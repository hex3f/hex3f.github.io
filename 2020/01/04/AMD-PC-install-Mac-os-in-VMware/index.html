AMD PC install Mac os in VMware

AMD平台在VMware安装mac os

其实最早之前是在VMware里装过mac的，但之前是intel的cpu，装mac会比较容易，问题也没那么多。最近去看了看unlocker的github项目好像已经删除了，应该是停止更新了。

环境：
DATE：2020/1/4 22:53
CPU：AMD R9 3900X
GPU: MSI RTX 2060 Super GAMING X
RAM: 32G
OS：Windows 10 Pro 18363
MacOS Ver：macOS.Mojave.10.14.4.x5 下载镜像 提取码：axib
VMware Ver：15.1.0 | 点击下载（试过15.5.1，但是装不上，最后会提示 未能启动虚拟机。并且如果系统是win10 1903或以上的版本，就需要使用15.1.0或以上版本，不然装不上。）
MK-unlocker | 点击前往下载页（这个版本是民间的一位大神改的，可以兼容15的版本，安装方法网上查就行了，一查一大把）
